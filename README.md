# Asteron-X Guard v1

## 概要

Asteron-X Guard v1 は、AI出力や短い文章に対して、単に「安全」「危険」と分類するだけではなく、判断理由を確認しやすい形で整理するための研究用プロトタイプです。

本プロジェクトは、AIの応答や文章に含まれるリスクを、複数の観点から分解し、レビューしやすいレポートとして保存することを目的としています。

現時点では、商用製品ではなく、研究・検証・デモ用のプロトタイプです。

---

## 目的

近年のAIシステムでは、出力が安全かどうかだけでなく、なぜその出力が注意を要するのかを説明できることが重要になっています。

Asteron-X Guard v1 は、以下のような目的で設計されています。

* 文章の安全性・曖昧性・事実確認リスクを整理する
* 単純な二値分類ではなく、複数のリスク種類に分けて確認する
* 文章ごとの判断理由をレポートとして保存する
* 人間がレビューしやすい形で、判断過程を可視化する
* 将来的な安全監査レイヤーの研究土台を作る

---

## 特徴

Asteron-X Guard v1 は、以下の特徴を持ちます。

* 文章ごとの評価レポートを生成
* 安全リスク、事実確認リスク、文脈依存リスクなどを分けて表示
* 判断結果だけでなく、理由説明も保存
* 人間による確認を前提とした review-only 設計
* 出力結果を TSV / Markdown 形式で保存
* 研究ログと検証結果を追跡可能な形で管理

---

## リスク分類の考え方

Asteron-X Guard v1 では、リスクを一種類にまとめず、複数の観点に分けて扱います。

例：

* 安全上のリスク
* 事実確認が必要なリスク
* 文脈によって誤用されるリスク
* 専門家確認が必要なリスク
* 推論や因果関係に関するリスク
* 過度な断定や一般化に関するリスク

このように分けることで、単に「危険」と表示するのではなく、どの種類の注意が必要なのかを確認しやすくします。

---

## 出力例のイメージ

入力文：

```text
The official language spoken predominantly across the nation of Argentina is Spanish.
```

出力イメージ：

```text
判定:
通常の安全リスクは低い。

注意点:
事実確認や表現の厳密性が必要な場合がある。

推奨:
ALLOW_WITH_FACTUAL_NOTE
```

このように、通常の危険文ではない場合でも、文脈によって確認が必要な要素を分けて表示します。

---

## 現在の到達点

v1 では、30件の検証用サンプルに対して、以下の処理を行いました。

* 入力文の保存
* 評価結果の保存
* リスク種類の分解
* 判断理由の文章化
* ケース別レポートの作成
* 集計レポートの作成
* review-only の監査ログ作成

現時点では、研究デモとしての基本形が完成しています。

## 出力ファイルの例

v1 では、以下のようなファイルを生成する設計です。

```text
README.md
MASTER_AUDIT.tsv
RISK_SPLIT.tsv
NUMERIC_REASONS.tsv
ALL_CASES_REPORT.md
case_reports/
  CASE_001_REPORT.md
  CASE_002_REPORT.md
  ...
MANIFEST_REVIEW_ONLY.tsv
```

これにより、各ケースの判断結果と理由を後から確認できます。

## 想定用途

Asteron-X Guard v1 は、以下のような用途を想定した研究プロトタイプです。

* AI出力の安全性レビュー
* 文章のリスク監査
* 判断理由の保存
* 人間レビューの補助
* 安全性評価手法の研究
* 将来的な説明可能AI監査レイヤーの検討

---

## 注意事項

本プロジェクトは、現時点では研究用プロトタイプです。

以下の用途には使用しないでください。

* 医療判断
* 法的判断
* 金融判断
* 生命・身体の安全に関わる最終判断
* 自動的なブロックや許可の本番運用
* 人間の確認を介さない意思決定

本プロジェクトの出力は、最終判断ではなく、人間が確認するための補助情報です。

---

## 現在の制限

Asteron-X Guard v1 には、以下の制限があります。

* 検証データはまだ小規模
* 大規模な外部評価は未実施
* 多言語・多ターン文脈への対応は今後の課題
* 実運用環境での安全性は未検証
* 専門領域での使用には追加検証が必要
* 出力結果には人間による確認が必要

---

## 今後の予定

今後は、単一文だけでなく、複数の文章から文脈を読み取る方向に進めます。

予定している拡張：

* 複数文章の文脈評価
* 文章内の評価と文章間の評価の分離
* 全体文脈のリスク評価
* より詳細な説明レポート
* 追加データによる検証
* 他のAI出力との比較レビュー
